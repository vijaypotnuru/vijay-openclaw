# Answer Overflow Skill

Search indexed Discord community discussions via Answer Overflow. Great for finding solutions to coding problems, library issues, and community Q&A.

## Installation

This is an instruction-only skill. Just place in your `skills/` directory.

## Usage

See SKILL.md for detailed usage instructions.

## Quick Example

```
# Search Answer Overflow
ollama_web_search "site:answeroverflow.com nextjs app router error"

# Fetch thread
ollama_web_fetch "https://www.answeroverflow.com/m/1234567890123456789"
```
